# MGCAMB_tests
Tests for MGCAMB

## 1. Requirements
MGCAMB has the same requirements as the Default CAMB 

## 2. Installing MGCAMB
Uses simply 
`$ make camb`

## 3. Running MGCAMB
Set the modified growth model and the Dark Energy model in params_MG.ini
Run the code as usual with
`$ camb params.ini`
